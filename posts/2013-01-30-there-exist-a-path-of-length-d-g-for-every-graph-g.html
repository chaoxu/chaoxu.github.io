There exist a path of length ⌈d(G)⌉ for every graph G


1 Unweighted Graph

G is a simple graph, then d(G)=∣G∣2e(G)​ be the average degree of a simple graph.

Lemma1

If G is a connected graph, then it contain a path of length min(2δ(G),∣G∣−1), where δ(G) is the minimum degree of G.

The above lemma is Exercise 1.7. in Graph Theory by Diestel.

Lemma2

Every graph G has a component H, such that d(H)≥d(G).

Proof

Fact: If xi​,yi​>0 and yi​xi​​<t for all 1≤i≤k, then ∑i=1k​yi​∑i=1k​xi​​<t. Assume the lemma is false, then consider all it’s components H1​,…,Hk​, d(Hi​)=∣Hi​∣2e(Hi​)​<d(G), then d(G)=∑i=1k​∣Hi​∣∑i=1k​2e(Hi​)​<d(G), a contradiction.

Theorem3

A simple graph G must contain a path of length at least d(G).

Proof

Proof by induction. It is true for graph with 1 vertex. Assume it is true for all graphs with k vertices, k<n. Consider graph G of n vertices. If the graph has more than 1 component, then we use Lemma 2 and show there exist a subgraph H with strictly smaller number of vertices, such that d(H)≥d(G), and just apply the induction hypothesis.

Otherwise, the graph is connected. If there exist a vertex v with degree at most 21​d(G), we can remove it, and d(G−v)≥d(G), then by inductive hypothesis, in d(G−v) there will be a path of length at least d(G). If there is no such vertex. then we must have δ(G)>21​d(G). By Lemma 1, we have it has a path of length min(2δ(G),∣G∣−1). 2δ(G)≥d(G) and d(G)≤∣G∣−1, thus it contain a path of length at least d(G).

2 Weighted Graph

Can we generalize this problem to weighted graphs? G=(V,E), (G,w) is a weighted graph, where w:E→R. Define the average weighted degree of graph G to be dw​(G)=∣G∣2∑e∈E​w(e)​, and the minimum weighted degree δw​(G)=e∈Emin​{w(e)}. If P is a path, then the weight of the path is W(P)=∑e∈P​w(e). What can we say about the path with maximum weight? It is easy to prove that there is a path of weight at least δw​(G).

Theorem4

For a weighted graph (G,w), there exist a path of weight at least δw​(G).

Proof

Consider the longest path v1​,…,vn​. Claim: w({vi​,vn​})≤w({vi​,vi+1​}) for all i. Assume not, then the path v1​,…,vi−1​,vi​,vn​,vn−1​…,vi+1​ would be heavier, a contradiction. Therefore we have δw​(G)≤{vi​,vn​}∈E∑​w({vi​,vn​})≤i=1∑n−1​w({vi​,vi+1​})=W(v1​…vn​)

However, we want something stronger, say instead of δw​(G), can it be dw​(G)? I have a proof but it uses a difficult lemma.

Definition5

A perfect path double cover (PPDC) for graph G is a set of paths, such that every edge is covered exactly two times, and every vertex is the end point of exactly two of the paths.

Note a PPDC for a graph with n vertices is a set of n paths.

Lemma6

Every simple graph has a perfect path double cover.

The lemma is proven by Hao Li in 1990.

Now we conclude with the theorem that eats up all the above special cases.

Theorem7

For a weighted graph (G,w), there exist a path of length at least dw​(G).

Proof

Consider a PPDC of G with n paths p1​,…,pn​. ∑i=1n​l(pi​)=ndw​(G), by pigeonhole principle, at least one of the path has length at least dw​(G).

I wonder if there is a simpler proof of the fact, so no high machinery like PPDC are involved.

All these results are of course, already known. Frieze, McDiarmid and Reed have solved it in 1992. It is a 10 page proof, and thanks to Charalampos who provided it in the comment. The proof for PPDC only used 2 pages, so this would be an easier proof.

How do we find such a path? PPDC can be computed in O(mΔ(G)) time, so we actually have a polynomial time algorithm for this.

This give us a fun problem.

Problem8

Prove that for every weighted multigraph with no self loop, if it has M edges and the largest set pairwise non-parallel edges has m edges, then there exist a path of weight at least M−m+1dw​(G)​.
