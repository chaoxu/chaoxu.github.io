Bounds on number of cuts


Recently there are some advances on counting the number of min-cut in graphs.

Consider we have an undirected graph G=(V,E) of n vertices, and there are positive cost c:E→R+ on the edges. We define c(F)=∑e∈F​c(e), to be the cost (value) of F⊂E.

Let P be a partition of V where each partition class is non-empty. We define E(P) to be the set of edges with end points in two different partition classes. A set of edges F is called a k-cut, if F=E(P) for some P such that ∣P∣≥k.

We stress that by this definition, a k-cut is a k−1-cut. A cut is a 2-cut. A min-k-cut is a k-cut of minimum value (cost). We let λk​ to denote the value of the min-k-cut.

It is well known that the number of min-cuts in a graph is (2n​)=O(n2). [1]

In the entire article, unless specifically stated, we assume k is a fixed integer at least 2, and α is a fixed value at least 1.

We can express the state alternatively.

Theorem1

The number of cuts F such that c(F)≤λ2​ is O(n2).

1 Bounds related to scaling of the min-cut

What happens when we want to know about the number of cuts with value at most αλ2​?

By simply analyzing Karger’s algorithm, one can obtain the following.

Theorem2

The number of cuts F such that c(F)≤αλ2​ is O(n2α).

With more careful analysis using tree packing, Karger obtained the following [2].

Theorem3

The number of cuts F such that c(F)≤αλ2​ is O(n⌊2α⌋).

Indeed, we do have a lower bound of (⌊2α⌋n​). Just consider an unweighted cycle, where min-cut has value 2. We can pick any ⌊2α⌋ edges, which forms a cut of value at most α times the min-cut.

Note that we require α to a fixed value. This is because there is a dependency on α hidden inside the big O. Our lower bound is absolute and does not depending on α being fixed. It be interesting to obtain a matching upper bound. Hence we can consider the problem with strict inequality.

Conjecture4

The number of cuts F such that c(F)<αλ2​ is O(n⌈2α⌉−1).

Henzinger and Williamson showed the conjecture true for all α≤23​ [3].

2 Bounds related to min-k-cut

There are multiple ways to obtain the following theorem. For example, directly generalize Karger’s argument for cut counting.

Theorem5

The number of cuts F such that c(F)≤λk​ is O(n2(k−1)).

There were many attempts, and people can only obtain lower bounds of the form (kn​). Again, a cycle would be an example of such lower bound. The min-k-cut has value k, and can be obtained by picking any k edges. The gap is pretty large. Hence one would tempt to conjecture the following.

Conjecture6

The number of cuts F such that c(F)≤λk​ is O(nk).

Recently, Gupta, Lee and Li almost closes the gap [4].

Theorem7

The number of cuts F such that c(F)≤λk​ is O^(nk).

Here O^ hides a factor smaller than any nϵ. While closing the gap, they showed thhe following interesting theorem.

Theorem8

The number of cuts F such that c(F)≤k(2−ϵ)λk​​ is O(n).

Conjecture9

The number of cuts F such that c(F)<k2λk​​ is O(n).

Note this theorem is basically shows we can also obtain interesting results for α=k2−ϵ​<1.

How about approximate min-k-cuts? Chekuri, Quanrud and I extended the tree packing analysis of Karger, and obtained the following result for k-cuts [5].

Theorem10

The number of cuts F such that c(F)≤αλk​ is O(n⌊α2(k−1)⌋).

Combining the Conjecture 6 and Conjecture 4, we get a unified conjecture, even for α<1.

Conjecture11

The number of cuts F such that c(F)<αλk​ is O(n⌈αk⌉−1).

3 Bounds on parametric cuts

Now, let’s consider parametric cuts. Consider we have d weight functions c1​,…,cd​:E→R≥0​. Define cμ​(e)=∑i=1d​μi​ci​(e). We interested in knowing about cuts F such that cμ​(F) is bounded by αλμ,k​, where λμ,k​ is the min-k-cut value when the cost function is cμ​.

Karger showed the following [6].

Theorem12

The number of cuts F such that cμ​(F)≤λμ,2​ for some μ∈R≥0d​ is O(nd+1).

A even more general theorem follows.

Theorem13

The number of cuts F such that cμ​(F)≤αλμ,k​ for some μ∈R≥0d​ is O(n2α(k−1)+d−1).

Hence, we would have the following conjecture.

Conjecture14

The number of cuts F such that cμ​(F)<αλμ,k​ for some μ∈R≥0d​ is O(n⌈αk⌉+d−2).

Note, we might relax the requirement that all ci​ and μ are non-negative. Aissi et. al. showed the following [7].

Theorem15

The number of cuts F such that cμ​(F)≤λμ​ for some cμ​≥0 is O(mdn2logd−1n).

The following would be even stronger conjecture.

Conjecture16

The number of cuts F such that cμ​(F)<αλμ,k​ for some cμ​≥0 is O(n⌈αk⌉+d−2).

4 Projected cut bounds

Let τe​=minU:e∈δ(U)​c(δ(U)). Fung et. al. showed a projected generalization of the cut counting bound [8]. Let Eλ​={e∣τe​≥x}.

Theorem17

The number of sets of the form F∩Eλ​ where F is a cut such that c(F)≤αλ is O(n2α).

If we let λ be the min-cut value, this is precisely the approximate cut counting bound.

We can of course ask if all our theorem can be applied to projected cuts. We don’t even know if it extends to k-cuts. However, we can expect the following ultimate conjecture.

Let τμ,k,e​ be the minimum over all cμ​(F), where F is a k-cut containing e. Let Eμ,k,λ​={e∣τμ,k,e​≥λ}.

Conjecture18

The number of sets of the form F∩Eμ,k,λ​ where F is a cut such that cμ​(F)<αλ for some cμ​≥0 is O(n⌈αk⌉+d−2).
