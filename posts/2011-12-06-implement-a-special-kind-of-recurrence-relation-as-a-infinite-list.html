Implement a special kind of recurrence relation as a infinite list


A common recurrence has the form an​={∑i=0∞​bi​an−mi​​cn​​if n>kif n≤k​ , where mi​ and bi​ are both infinite sequences, and ci​ is a finite sequence. mi​∈N. a−i​=0 for all positive i. This is well defined as long as bi​,ai​,ci​ are in the same ring.

One can use a balanced binary tree to store the entire infinite list, and the time to generate the nth element is O(d(n)logn), where d is the density function of {mi​}.

Using an array would make it O(d(n)), but it is too imperative for our taste, how about we only use list and achieve O(d(n)) time, elegantly?
