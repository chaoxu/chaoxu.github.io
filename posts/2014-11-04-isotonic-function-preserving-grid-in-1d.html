Isotonic function preserving grid in [0,1]


Definition1

A function φ:X→Y is isotonic if ∥x−y∥<∥w−z∥⟹∥φ(x)−φ(y)∥<∥φ(w)−φ(z)∥ for all x,y,z,w∈X.

A sequence of points x0​,x1​,…,xn​ is called δ-grid if δ<5n1​, x0​=0, xn​=1 and for all 1≤i≤n−1, we have xi​−xi−1​∈(n1​−2i−1δ​,n1​−2iδ​). Note this imply xn​−xn−1​>n1​.

Theorem2

Let X be a δ-grid of n points where δ<2−n. φ:X→[0,1] is a isotonic function such that φ(0)=0 and φ(1)=1, then ∣xi​−φ(xi​)∣<1/n

Proof

It’s easy to see that φ is a increasing function. Let the points in X ordered as x0​,x1​,…,xn​. Let li​=∣φ(xi​)−φ(xi−1​)∣. Note that

  1. i=1∑n​li​=1

  2. ∣xi​−xi−1​∣<n1​−2iδ​<∣xi+1​−xi​∣ thus by isotonic function ∣φ(xi​)−φ(xi−1​)∣<∣φ(xi+1​)−φ(xi​)∣. This is just li​<li+1​.

  3. ∑i=1m​li​>∑i=n−(m−2)n​li​ for all m, because nm​≥∣xm​−x0​∣>nm​−δ>nm−1​+2δ>∣xn​−xn−(m−2)​∣, thus ∣φ(xm​)−φ(x0​)∣<∣φ(xn​)−φ(xn−(m−2)​)∣.

Combine the relations above, we have nm​≥i=1∑m​li​>i=n−(m−2)∑n​li​≥nm−1​

But ∑i=1m​li​=φ(xm​), so m/n≥xm​>m/n−δ, m/n≥φ(xm​)>(m−1)/n. Since δ is small, we have ∣xi​−φ(xi​)∣<1/n.
